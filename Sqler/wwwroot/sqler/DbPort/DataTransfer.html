数据直导
数据源配置 指定要从中复制数据的源:
type mssql mysql sqlite
connectionString
sql(若不指定则复制所有表数据)
目标配置 指定要将数据复制到何处:
type mssql mysql sqlite
connectionString
createTable
delete
truncate
            
                type:               数据库类型，可为 mssql、mysql、sqlite
                ConnectionString:   数据库连接字符串
                createTable:        若指定(on)，则在导入数据前自动创建表结构
                delete:             若指定(on)，则在导入数据前使用delete语句清空待操作的表
                truncate:           若指定(on)，则在导入数据前使用truncate语句清空待操作的表
